# velocitymotors
Wireframe has been added to the proposal that was approved.

1. Project Overview
This project forms the final phase of the Velocity Motors website development. Part 3 focuses on adding dynamic JavaScript functionality, enhancing user interaction, implementing search and filter features, creating interactive maps, validating forms, and applying SEO best practices.

2. JavaScript Functionality Implemented
2.1 Dynamic Car Listings
- Cars are generated using JavaScript arrays.

2.2 Search Functionality
- Filters vehicles by model name.

2.3 Price Filter
- Filters vehicles by price range.

2.4 Lightbox Gallery
- Enlarges images in a popup.

2.5 Interactive Map
- Leaflet map with Pretoria branche.

2.6 Form Interactivity
- Enquiry and contact forms with validation and mailto functionality.

3. SEO Implementation
3.1 On-Page SEO
- Title tags, meta descriptions, keywords, alt text, structured headings.

3.2 Off-Page SEO
- Backlink strategy and social media sharing.

3.3 Technical SEO
- robots.txt, sitemap.xml, deferred scripts, optimized images.
